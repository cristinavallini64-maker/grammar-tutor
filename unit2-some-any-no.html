🔵 SOME / ANY / NO
1 / 4
✅ SOME — frasi AFFERMATIVE

Some = un po' di / alcuni-e. Si usa nelle frasi affermative con C plurali e con U.

some
C'è del formaggio nel frigo. (cheese = U)
some
Ci sono dei pomodori nell'insalata. (tomatoes = C)
some
Vorrei un po' di torta, per favore.
offerta
Vuoi un po' d'acqua? (offerta → some anche in domanda!)
Some si usa anche nelle domande-offerta/richiesta:
Can I have some bread, please?  ·  Would you like some tea?
❓ ANY — frasi NEGATIVE e INTERROGATIVE

Any = del/della/dei/delle. Si usa nelle frasi negative e interrogative.

any ❓
Ci sono dei pomodori?
any ❓
C'è dello zucchero?
any ✗
Non ci sono carote.
any ✗
Non c'è olio d'oliva. (olive oil = U)
⚠️ Domanda con any: risposta affermativa → usa some!
— Are there any eggs? — Yes, there are some.
🚫 NO — significato negativo DIRETTO

No = nessuno/a, niente. Va con verbo affermativo (la negazione è nel determinante stesso).

no
Non c'è zucchero nel caffè.
no
Equivalente a: There aren't any carrots.
no
Non possiamo fare un panino. Non c'è pane.
⚠️ No va con verbo affermativo — già contiene la negazione!
There isn't no sugar. ✗ (doppia negazione!) → There is no sugar. ✓
oppure → There isn't any sugar. ✓
🔄 SOME / ANY / NO — RIASSUNTO
SOME
There is some milk. / Would you like some?
ANY
Is there any milk? / There isn't any milk.
NO
There is no milk. (= There isn't any milk.)
C / U?
some rice (U) · some apples (C) · any water (U) · any eggs (C)
💡 Trucco: some = +  ·  any = ? / −  ·  no = − (senza not!)
Se vedi not nel verbo → usa any. Se verbo è positivo e vuoi negare → usa no.
