📋 CHE COS'È IL FIRST CONDITIONAL?
Il first conditional ("periodo ipotetico di primo tipo") si usa per parlare di situazioni reali o molto probabili nel futuro. Si forma con if + present simple nella condizione e will + verbo base nella conseguenza. Il risultato è possibile o probabile se la condizione si avvera.

⚡ Nella condizione con if NON si usa will — solo il present simple! La virgola va dopo la condizione quando viene prima: If it rains, I'll call you. / I'll call you if it rains.
🔧 ESEMPI — situazioni future possibili
POSITIVA Se restiamo a casa, guarderemo un film.
NEGATIVA Se non restiamo a casa, non guarderemo un film.
MISTA Se siamo stanchi, non usciremo.
ORDINE INVERTITO Ci diranno se hanno notizie. (niente virgola)
CON WHEN Quando avranno notizie, ce lo diranno. (when = certezza)
⚠️ Non si può usare if al posto di when nel first conditional in inglese: Will Rory be happy if he gets a puppy? (NON if he'll get)
✂️ STRUTTURA — tutte le combinazioni
Se restiamo a casa, guarderemo un film.
Se non siamo stanchi, usciremo.
Se piove, non andrò in spiaggia.
Se non ci sbrighiamo, non prenderemo il bus.
⚡ Se la parte con if viene prima → virgola. Se viene dopo → niente virgola. If I'm late, I'll call you. / I'll call you if I'm late.
